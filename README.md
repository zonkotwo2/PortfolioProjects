# Data Analyst Portfolio

Graduate Data Analyst portfolio showcasing projects in SQL, Power BI, data analysis, machine learning and technical problem-solving.

This portfolio demonstrates my ability to clean messy datasets, query data using SQL, build dashboards, analyse trends, communicate findings clearly and apply technical thinking to real engineering and data problems.

## Projects

### 1. SQL Data Cleaning and Analysis

Folder: `SQL-Data-Cleaning-and-Analysis`

This project uses SQL to clean and analyse a layoffs dataset. The aim was to transform raw, inconsistent data into a structured dataset suitable for analysis, then use SQL queries to identify trends by company, industry, country and time period.

**Skills shown:**  
SQL, MySQL, data cleaning, staging tables, duplicate removal, text standardisation, date cleaning, null handling, aggregations, trend analysis and business insight.

**Key areas covered:**
- Created staging tables to preserve raw data
- Removed duplicate records
- Standardised inconsistent text values
- Cleaned date formats
- Reviewed null and blank values
- Analysed layoff trends by company, industry, country and time period

### 3. Human Brain Sensing Using Photonics Thesis

File: `ThesisHandIn.pdf`

This is my final Biomedical Engineering dissertation, focused on using laser speckle imaging and machine learning to classify brain states from non-invasive biomedical imaging data.

The project involved raw data processing, feature engineering, model comparison and technical reporting. I investigated different processing and modelling approaches, then communicated the results through a full technical thesis.

**Skills shown:**  
Machine learning, data processing, feature engineering, model evaluation, technical reporting, independent research, problem-solving and scientific communication.

**Key areas covered:**
- Processed complex biomedical imaging data
- Engineered features from raw signal/image data
- Compared machine learning approaches
- Evaluated model performance
- Communicated findings through a detailed technical report


### 4. Cellexus Bioreactor Project Contributions

My contributions included C++ coding for different bioreactor systems and pump control, producing SolidWorks parts, and supporting practical engineering development across circuitry, design and testing outputs.

Although this folder is named `Cellexus-Bioreactor-Project`, it represents my individual project outputs and contributions to the Cellexus bioreactor project.

**Skills shown:**  
C++, embedded systems thinking, pump/control logic, SolidWorks, CAD design, circuitry, prototyping, technical problem-solving and engineering documentation.

**Key areas covered:**
- Developed C++ code for bioreactor system functions
- Worked on pump/control logic
- Designed and produced parts using SolidWorks
- Contributed to circuitry and practical engineering outputs
- Supported the development of a Luer-Lock/bioreactor solution
- Documented individual technical contributions from the group project
