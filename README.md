# Data Analyst Portfolio

Graduate Data Analyst portfolio showcasing projects in SQL, Power BI, Excel and data analysis.

These projects demonstrate my ability to clean messy datasets, query data using SQL, build dashboards, analyse trends and communicate findings clearly for decision-making.

## Projects

### 1. SQL Data Cleaning and Analysis

Cleaned and analysed a global layoffs dataset using MySQL.  
The project involved creating staging tables, removing duplicates, standardising text fields, cleaning date formats, handling null values and writing analysis queries to identify layoff trends.

**Skills shown:** MySQL, SQL, data cleaning, staging tables, duplicate removal, aggregations, trend analysis, business insight.

### 2. Power BI Dashboard

Built an interactive Power BI dashboard using Power Query, relationships, DAX measures and visual storytelling.  
The dashboard was designed to summarise key performance metrics and make trends easier to interpret.

**Skills shown:** Power BI, Power Query, DAX, data modelling, KPI reporting, dashboard design, data visualisation.

### 3. Human Brain Sensing Thesis Project

Contains original outputs from my Biomedical Engineering dissertation on non-invasive brain sensing using photonics.  
The project involved processing complex imaging data, comparing machine learning methods and presenting clear technical conclusions.

**Skills shown:** data processing, feature engineering, model evaluation, technical reporting, independent research.
